Dieses **NBA Players Stats Dataset Sample** umfasst **1.000 Datensätze** und wurde mithilfe der **Bright Data API** extrahiert.

## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | Die ESPN-URL, die direkt zur Profilseite des Spielers mit detaillierten Informationen führt. |
| `player_name` | Der vollständige Name des NBA-Spielers. |
| `player_games_played` | Gesamtzahl der Spiele, an denen der Spieler während der Saison teilgenommen hat. |
| `player_games_started` | Gesamtzahl der Spiele, die der Spieler in der Saison von Beginn an gespielt hat. |
| `player_minutes_per_game` | Durchschnittlich gespielte Minuten pro Spiel des Spielers. |
| `player_points_per_game` | Durchschnittlich erzielte Punkte pro Spiel des Spielers. |
| `player_offensive_rebounds__per_game` | Durchschnittliche Anzahl offensiver Rebounds pro Spiel. |
| `player_defensive_rebounds_per_game` | Durchschnittliche Anzahl defensiver Rebounds pro Spiel. |
| `player_rebounds_per_game` | Durchschnittliche Gesamtzahl der Rebounds pro Spiel des Spielers. |
| `player_assists_per_game` | Durchschnittliche Anzahl der Assists pro Spiel des Spielers. |
| `player_steals_per_game` | Durchschnittliche Anzahl der Steals pro Spiel des Spielers. |
| `player_blocks_per_game` | Durchschnittliche Anzahl der Blocks pro Spiel des Spielers. |
| `player_turnovers_per_game` | Durchschnittliche Anzahl der Turnovers pro Spiel des Spielers. |
| `player_fouls_per_game` | Durchschnittliche Anzahl der pro Spiel begangenen Fouls des Spielers. |
| `player_assist_to_turnover_ratio` | Verhältnis von Assists zu Turnovers des Spielers. |
| `team` | Das NBA-Team, dem der Spieler in dieser Saison zugeordnet ist. |
| `season_type` | Gibt an, ob die Daten aus der regulären Saison oder aus den Playoffs stammen. |
| `season_year` | Das Jahr der NBA-Saison, aus der die Statistiken stammen. |


## Use Cases

### 1. Analyse der Spielerleistung
Analysieren Sie die Leistung einzelner Spieler über verschiedene Saisons hinweg, um Top-Performer zu identifizieren und Teamstrategien zu verbessern.

### 2. Fantasy-Basketball-Insights
Nutzen Sie historische Spielerstatistiken, um Insights für Fantasy-Basketball-Ligen zu generieren und die Spielerauswahl sowie Vorhersagen zu unterstützen.

### 3. Sportjournalismus und Berichterstattung
Liefern Sie fundierte statistische Analysen für Sportartikel und bereichern Sie Ihre Berichterstattung mit datengestützten Insights zur Spielerleistung.
